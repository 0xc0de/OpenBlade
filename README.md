# OpenBlade

Open source re-implementation of Blade Of Darkness.

Work in progress.

Build

1. Clone repository with all submodules: git clone --recurse-submodules https://github.com/0xc0de/OpenBlade
2. Use Cmake to configure/generate/build project

Install

1. Install the original game on your computer.
2. Open Data/default.cfg and specify startup settings, including path to the original game. E.g: "C:\Games\Blade Of Darkness"
3. Run OpenBlade.exe.
